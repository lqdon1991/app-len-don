# Nutrilite - Lên Thực Đơn Dinh Dưỡng

Ứng dụng web giúp tạo thực đơn dinh dưỡng cá nhân hóa sử dụng sản phẩm Nutrilite của Amway dựa trên thói quen ăn uống và sinh hoạt.

## Tính năng

- Đánh giá sức khỏe thông qua các câu hỏi về thói quen sinh hoạt
- Đề xuất sản phẩm Nutrilite phù hợp dựa trên:
  - Tuổi tác
  - Giới tính
  - Thói quen ăn uống
  - Lối sống
  - Mục tiêu sức khỏe

## Công nghệ

- React 18
- TypeScript
- Vite
- Tailwind CSS

## Cài đặt

```bash
npm install
```

## Chạy ứng dụng

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy lên Vercel

1. Đẩy code lên GitHub
2. Kết nối repository với Vercel
3. Vercel sẽ tự động detect và deploy

Hoặc sử dụng Vercel CLI:

```bash
npm i -g vercel
vercel
```
